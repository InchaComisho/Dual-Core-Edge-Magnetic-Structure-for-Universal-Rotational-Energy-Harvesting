## 概要

このリポジトリは、**汎用回転エネルギー回収のためのデュアル・コアエッジ磁気構造**を記録するための公開技術文書です。

この構想は、風、水流、波、潮流、排水、人工的なスパイラル流など、自然界またはインフラ内に存在する回転・流体エネルギーを、モジュール型の発電ユニットで電力へ変換することを目的としています。

基本構造は、**外周の回転磁石リング**と、**中心の固定コイル／コア軸**を組み合わせたデュアル構成です。外側の磁石リングが回転することで磁束が変化し、中心の固定コイル側で電磁誘導により電力を取り出します。

これは**オープン発明の先行公開文書**であり、実証済みの工業製品ではありません。永久機関、フリーエネルギー、エネルギー保存則の突破、無条件の高効率化を主張するものではありません。出力は常に、入力される機械的エネルギー、負荷条件、磁気損失、電気損失、軸受損失、発熱、制御回路の効率に制約されます。

目的は、この構想を検索可能・引用可能・検証可能な形で公開し、試作、批判、改善、オープンなエネルギー開発につなげることです。

## 基本構想

この発電ユニットは、主に次の二つの磁気領域で構成されます。

1. **外周磁石リング**  
   円周上に永久磁石を配置した回転構造。ローターが回転することで、固定コイル／コアに対して磁束変化を発生させます。

2. **中心固定コイル／コア軸**  
   コイル、磁性コア、積層鉄心などを持つ固定構造。変化する磁束を電力へ変換します。

3. **デュアル・コアエッジ磁気相互作用**  
   内外の磁気領域を、エッジまたは環状の界面で磁束変化が強く現れるように配置します。

4. **非接触型に近いエネルギー抽出領域**  
   発電部では、ギアのような直接的な機械接触を減らし、磁気的な結合で回転エネルギーを電気へ変換することを目指します。ただし、軸受、支持部、シール、流体接触部などの機械要素は依然として必要です。

5. **汎用回転インターフェース**  
   同じ発電コアを、垂直軸風車、小水力ローター、潮流タービン、波力変換機構、排水タービン、人工スパイラル流路などに接続できる可能性があります。

## 物理的基礎

この構想は標準的な電磁誘導に基づきます。

コイルに誘導される電圧は、ファラデーの法則に従います。

```text
V = -N * dPhi/dt
```

ここで、

- `V` = 誘導電圧
- `N` = コイル巻数
- `Phi` = コイルを貫く磁束
- `dPhi/dt` = 磁束の時間変化率

入力される機械的パワーは、次で表されます。

```text
P_mechanical = torque * angular_velocity
```

出力電力は、常に以下の制約を受けます。

```text
P_output <= P_mechanical - losses
```

重要な点：

- 入力回転エネルギーなしに発電することはできません。
- 電気負荷をかけると、レンツの法則により反トルクが発生します。
- 磁気設計により結合を改善し損失を減らすことはできますが、エネルギー保存則を回避することはできません。
- 効率は、トルク、RPM、電圧、電流、力率、温度、負荷曲線などを測定して検証する必要があります。

## 応用可能領域

### 1. 垂直軸風力

垂直軸風車に外周磁石リングを接続することで、低高度の都市部や農村部でも、全方向の風を利用した小規模発電を試せます。

### 2. 小水力・排水流

農業用水路、建物排水、制御された水流、小河川などに設置し、連続的な低速回転エネルギーを回収する可能性があります。

### 3. 潮流・河川流

潮流や河川流は持続的な運動エネルギーを持ちます。耐腐食設計を行えば、海水・汽水環境での試作も考えられます。

### 4. 波力変換

波の運動は直接回転ではありませんが、浮体、クランク、ラチェット、直線運動から回転運動への変換機構、油圧機構などを通じて回転に変換できます。

### 5. 人工スパイラル流

人工水路、圧力差、排水、循環水路、冷却システムなどに存在する流れを、スパイラル流や小型タービンで回転に変換して発電する可能性があります。

## 検証すべき期待効果

以下は仮説であり、実証済みの性能主張ではありません。

1. **発電部の摩耗低減**  
   ギア中心の構造よりも、磁気的な非接触結合を用いることで摩耗を減らせる可能性があります。

2. **複数の流体源への共通利用**  
   同一の発電コアを、異なるローター形状と組み合わせられる可能性があります。

3. **保守性の向上**  
   コイル／コアが固定されていれば、電気系の保守が容易になる可能性があります。

4. **低速回転への最適化可能性**  
   高極数、強いエッジ磁束、適切な巻線設計により、低RPMでも有効な電圧を得られる可能性があります。

5. **分散型電源との相性**  
   小型ユニットは、マイクログリッド、センサー、散水、非常灯、オフグリッド監視などに利用できる可能性があります。

## この構想が主張しないこと

このリポジトリは、次を主張しません。

- 永久機関
- フリーエネルギー
- 入力なしの発電
- レンツの法則の無効化
- すべての既存タービンに対する無条件の高効率
- 完全な摩擦ゼロ
- 完全な保守不要
- 試作なしの産業実装可能性
- 実証済み商用性能

これは、物理試作、測定、批判、改善を必要とする**オープン発電構造の概念提案**です。

## 試作ロードマップ

### Phase 0 — シミュレーションと設計

- 磁石数、極配置、コイル形状、空隙を定義する。
- 可能であれば有限要素法による磁場解析を行う。
- RPMに対する開放電圧を推定する。
- 負荷接続時の出力をモデル化する。
- 起動トルクと抵抗を見積もる。

### Phase 1 — 卓上試作

- 手回し、またはドリル回転による小型試作機を作る。
- 電圧、電流、RPM、トルク、発熱、負荷応答を測定する。
- 単一コア構造とデュアル・コア構造を比較する。
- デュアル・コアエッジ構造が有効な磁束変化を増やすか記録する。

### Phase 2 — 小水力 / 送風試験

- 小型の水流または風流ローターに接続する。
- 制御された流速下で発電量を測定する。
- 同程度のサイズ・コストの既存発電機と比較する。

### Phase 3 — フィールド試験

- 農業用水路、排水、屋上風力、波力水槽などに設置する。
- 耐久性、腐食、振動、異物混入、保守性を観察する。

### Phase 4 — オープンハードウェア化

- CAD、巻線仕様、磁石配置、測定データ、失敗例を公開する。
- 独立した再現実験を受け入れる。

## 最低限必要な測定項目

信頼できる試験では、次を記録する必要があります。

| 測定項目 | 意味 |
|---|---|
| RPM | 磁束変化の周波数を決める |
| トルク | 機械的入力を計算するために必要 |
| 電圧 | 電気出力の可能性を示す |
| 電流 | 実際に取り出せる電力を示す |
| 負荷抵抗 | 負荷曲線解析に必要 |
| 温度 | コイル損失と熱限界を示す |
| 空隙 | 磁気結合に大きく影響する |
| 磁石グレード・極数 | 磁場強度と周波数を決める |
| 巻数・線径 | 電圧と電流の特性を決める |
| 効率 | 入力に対してどれだけ出力できたかを示す |

推奨式：

```text
効率 = 電気出力 / 機械入力
```

```text
電気出力 = 電圧 * 電流
機械入力 = トルク * 角速度
```

## 自然補完科学・人工叡智との関係
