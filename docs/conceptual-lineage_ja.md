# 構想の系譜：火力発電所改修案から汎用回転エネルギー回収へ

## 元NOTE記事群と設計思想の発展

このリポジトリは、単独の新型発電機構想ではありません。**汎用回転エネルギー回収のためのデュアル・コアエッジ磁気構造**は、火力発電所改修、逆位相コイル、モーター内エネルギー回収、垂直軸タービン発電、エネルギー循環構造に関する複数の日本語NOTE記事から発展したものです。

現在のGitHub版は、それらの初期構想を、英語圏でも読める技術文書・オープン発明資料として整理し直したものです。

---

## 1. 原点：火力発電所のCO2排出削減

初期の問題意識は、単に新しい再生可能エネルギー発電機を作ることではありませんでした。

原点は、既存の火力発電所を改修し、化石燃料依存とCO2排出負荷を下げるための補助的なエネルギー回収システムです。

初期モデルでは、次のような二次的エネルギー回収層が考えられていました。

- 渦巻き / スパイラル状の水流
- ブレードによる回転
- 回転磁石
- 固定コイル
- 整流
- 変圧 / インバーター
- バッテリー
- 補助送電

この原点モデルでは、既存の火力発電インフラを即座に廃棄するのではなく、内部に存在する排水、冷却水、蒸気関連の流れ、人工的な渦流などを、回収可能な回転エネルギー源として扱います。

構想の流れは次の通りです。

```text
火力発電所改修
-> 渦巻き / スパイラル流の回収
-> 回転磁気発電
-> バッテリー蓄電
-> 補助送電
-> 汎用回転エネルギー回収
```

## 4. 構想の発展段階

この構想の発展は、次のように整理できます。

```text
1. モーター回転や逆起電力的挙動からのエネルギー回収
2. 逆位相 / 発電側コイルの追加
3. 整流・昇圧・安定化・蓄電
4. 火力発電所改修システムへの応用
5. 渦巻き / スパイラル流と垂直軸回転の利用
6. 中心軸 + 外周リングによるデュアル・コアエッジ磁気発電
7. 風・水・潮流・波・排水・人工流体流への汎用化
```

現在のリポジトリは、この第7段階、すなわち汎用回転エネルギー回収構造として整理したものです。

## 5. 技術的注意

元NOTE記事群は、初期段階のオープン発明提案を含みます。したがって、実証済み性能の主張として読むべきではありません。

GitHub技術文書としての安全な位置づけは次の通りです。

- 構想：オープン設計仮説として有効
- 物理基礎：電磁誘導と回転エネルギー変換
- 性能：測定されるまで未検証
- 効率：機械入力と電気出力から計算する必要がある
- 主張：永久機関、フリーエネルギー、無条件高効率、入力なし発電は主張しない

## 6. なぜこの系譜が重要なのか

この系譜を記録する意味は、この構想が抽象的な新型発電機から始まったのではないと示す点にあります。

原点は、現実的な移行問題への問いでした。

> 既存の火力発電所や産業インフラの中で失われている回転、熱、水流、空気流を、廃棄せずに局所的に回収できないか？

この原点は、自然補完科学の原則と一致します。

```text
新しい負荷を作る前に、既に存在する流れを使う。
新しい燃料を求める前に、既に動いているものを回収する。
インフラを捨てる前に、改修して補完する。
性能を主張する前に、測定する。
```
